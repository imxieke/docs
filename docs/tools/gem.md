## Usage
- `gem sources --add https://mirrors.aliyun.com/rubygems/`
- `gem sources --remove https://rubygems.org`
- `gem sources --remove https://rubygems.org/`
- `bundle config mirror.https://rubygems.org https://mirrors.aliyun.com/rubygems/`
